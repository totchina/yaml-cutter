# Yaml-Cutter Developer Documentation

#### Setup the development environment
* install Node.JS
* `setup`


#### Run tests

* run all tests

  ```
  $ bin/spec [test file]
  ```

* run only unit tests

  ```
  $ bin/tests
  ```

* run only Cucumber

  ```
  $ bin/cuke
  ```


#### Update dependencies

```
$ update-check
$ update
```


#### Release a new version

```
$ publish <patch|minor|major>
```
